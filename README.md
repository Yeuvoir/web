## 运行方式 (Maven)

**编译打包：**
```bash
mvn clean package
```

**启动服务器：**
```bash
java -jar target/simple-http-socket-1.0-SNAPSHOT.jar server 8080[可选]
```

**启动客户端 GUI：**
```bash
java -jar target/simple-http-socket-1.0-SNAPSHOT.jar client
```
